# ChitChat - Available Commands

| # | Command | Example |
|---|---------|---------|
| 1 | register | `register Fname Lname username password` |
| 2 | login | `login username password` |
| 3 | friend | `friend username` |
| 4 | logout | `logout` |
| 5 | createchat | `createchat user1 user2 ...` |
| 6 | chat | `chat <convo_id> <message>` |
